# LLM Benchmarks

Base Astro para um portfolio visual sobre benchmarks de modelos de IA.

## Estrutura

- `src/pages/`: rotas Astro
- `src/layouts/`: layouts compartilhados
- `src/components/`: blocos reutilizáveis da interface
- `src/data/`: conteúdo, taxonomias e traduçőes
- `src/styles/`: estilos globais

## Comandos

```bash
npm install
npm run dev
```
